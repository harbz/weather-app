# Weather App 🌤️

This is a simple weather app built using HTML, CSS, and JavaScript.  
It fetches current weather data using the OpenWeatherMap API.

## Features
- Current temperature
- City search
- Responsive layout

## How to Use
1. Enter a city name.
2. View the current weather.

## API
[OpenWeatherMap API](https://openweathermap.org/api)
